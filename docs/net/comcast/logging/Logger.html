Packagenet.comcast.logging
Classpublic class Logger

The Logger class logs messages from the application to a console. The Logger class itself is only responsible for determining whether the message should be logged based on the level of the message. The formatting and output is handled by the console.Loggers can be instantiated with different Consoles (see TraceConsole, FlexConsole etc)
